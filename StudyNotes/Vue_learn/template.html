过滤器

过滤器函数接受表达式的值作为第一个参数

{{ message | capitalize }}

当然，过滤器是函数，可以接受其他参数的。比如我们写成{{ message | filterA('arg1', arg2) }}，message自动是第一个参数，然后'arg1'和arg2作为第二个参数和第三个参数

过滤器还可以串联，形如这样{{ message | filterA | filterB }}
