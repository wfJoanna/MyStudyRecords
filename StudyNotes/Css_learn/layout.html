CSS 元素居中对齐

水平居中块级元素 (如 div), 可以使用 margin: auto;

CSS 文本居中对齐

如果仅仅是为了文本在元素内居中对齐，可以使用 text-align: center;

CSS 图片居中对齐

要让图片居中对齐, 可以使用 margin: auto;

CSS 左右对齐-使用定位方式

我们可以使用 position: absolute; 属性来对齐元素

右边的那块儿没有设置top与bottom，所以上下位置与原文档流位置一致。但是是不包含在文档流中的，所以会和别的元素重叠

CSS 左右对齐-使用浮动方式

我们可以使用 float 属性来对齐元素。记得clear

你会发现float和position和浏览器右边缘距离不同。只要给body设置padding为0、margin为0，float就也能紧贴浏览器窗口右边缘咯

CSS 垂直居中-使用padding

使用 padding 属性实现元素的垂直居中


如果要水平和垂直都居中，可以使用 padding 和 text-align: center
