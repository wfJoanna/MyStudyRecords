CSS resize

有了resize属性，用户就可以调整元素大小

resize: none|both|horizontal|vertical;
none，用户无法调整元素的尺寸。（默认）
both，用户可调整元素的高度和宽度。
horizontal，用户可调整元素的宽度。
vertical，用户可调整元素的高度。

经测验，图片好像没法直接resize，但是可以放到div中（不过也并不能改变图片大小，只能改变显示范围）
