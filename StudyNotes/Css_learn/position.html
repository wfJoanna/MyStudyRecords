CSS position:static; 静态定位

static就是默认值，即没有定位，遵循正常的文档流对象
即使设定了top、left、这些，也不会产生影响
本大段使用了position:static;

CSS position:fixed; 固定定位

元素的位置相对于浏览器窗口是固定位置。即使窗口是滚动的它也不会移动
fixed定位使元素的位置与文档流无关，因此不占据空间
fixed定位的元素会和其他元素重叠

CSS position:relative; 相对定位

相对定位元素的定位，是相对其正常位置而言的

元素会移动，但它原本所占的空间不会改变

这是位于正常位置的标题

这个标题相对于其正常位置向左移动

这个标题相对于其正常位置向右移动

这是一个没有定位的标题

这个标题是根据其正常位置向上移动

CSS position:absolute; 绝对定位

绝对定位的元素的位置，是相对于最近的已定位父元素（static除外）。注意是父元素
如果元素没有已定位的父元素，那么它的位置相对于html，初始位置就跟fixed一样了，只不过会随着滚动而移动
absolute定位使元素的位置与文档流无关，因此不占据空间
absolute定位的元素会和其他元素重叠

这是一个绝对定位了的标题

CSS position:sticky; 粘性定位

基于用户的滚动位置来定位（具体的复杂得很，总之先记住这一种效果吧）

我是粘性定位!

CSS 重叠问题

元素的定位与文档流无关，所以它们可以覆盖页面上的其它元素
z-index属性指定了一个元素的堆叠顺序，具有更高堆叠顺序的元素总是在较低的堆叠顺序元素的前面
如果两个定位元素重叠，没有指定z-index，最后定位在HTML代码中的元素将被显示在最前面

因为图像元素设置了 z-index 属性值为 -1, 所以它会显示最后。
